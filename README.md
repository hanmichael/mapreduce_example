## 本项目 为常用MapReduce算法的 几种实现

01.词频计算------------------------------WordCount

02.二次排序------------------------------AmtSort

03.数据清洗------------------------------DataClean

04.数据去重------------------------------Distinct

05.join操作-----------------------------Join

06.倒排索引------------------------------Index

07.找共同好友----------------------------FindFriend

08.求平均数------------------------------Average

09.求附近地标----------------------------NearbyCoordinate

所需要的输入文件在InputResource文件夹中
产生的输出文件在Output中(每次执行,若存在,会自动删除)

为了简单展示,文件路径都为 本地路径
